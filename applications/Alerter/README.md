# Alerter
A simple dialog box with JavaFX (version 11)

## Requirements
- Java SE Development Kit (tested on 11.0.1)
- Apache Maven (tested on 3.6.0)

## Building
`$ mvn clean package`

## Running
`$ java -jar ./target/alerter-1.0.1-jar-with-dependencies.jar --title="Message" --content="Hello World!"`
